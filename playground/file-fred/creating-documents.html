Creating Documents

Fred is also able to create documents based on a text input. In this demonstration you will see the speed and accuracy with which File Fred generates a quote based on a customer request in an incoming email.

To test Fred's abillities, press submit.

Input:
Hey Pete,

I hope this email finds you well.
I would like to order again from your store.

I would like to have:
1. One pepe jeans in tapered fit mid rise in blue
2. One mosaic design boys shirt in size 36
3. One men shirt, striped blue in size 42

Please bill and ship to the address I used last time.
Thank you very much.

Kind regards,
Melba Scott
